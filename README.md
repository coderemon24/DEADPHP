# 🔥 DEADPHP — Custom PHP MVC Framework

**DEADPHP** is a lightweight, **custom MVC framework** built with pure PHP — no composer, no Laravel, no dependencies.

Designed for:
- Beginners learning MVC
- Developers who want a minimal PHP framework
- Fast and clean custom routing
- Structured app architecture without heavy libraries

---

## 🚀 Overview

DEADPHP is a **basic MVC framework from scratch** with:

✔ Routing  
✔ Controller Base  
✔ Models  
✔ Views (Blade-like structure)  
✔ Config system  
✔ Database connection support  
✔ Simple helper system

This project is perfect for learning how the MVC pattern works internally — **no external dependencies required!**

---

## 🛠 How To Use

```
git clone https://github.com/coderemon24/DEADPHP.git
cd DEADPHP
php -S localhost:8000 -t public          # start server
http://localhost:8000                    # visit actual project
```
